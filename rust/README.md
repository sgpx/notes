# rustup

rust toolchain installer

https://forge.rust-lang.org/infra/other-installation-methods.html

# install

```
wget -v https://static.rust-lang.org/rustup/dist/aarch64-unknown-linux-gnu/rustup-init
chmod +x rustup-init
./rustup-init
```
