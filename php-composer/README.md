# composer

php package manager

# setup (ubuntu)

`apt install -y composer`

# package registry

packagist.com

# init

`composer init --quiet`

or 

`composer init -q`

# install package

`composer require rmccue/requests`

# invoke package

`require "/path/to/vendor/mypackage/foo.php"`

`include "/path/to/vendor/mypackage/foo.php"`

`require_once "/path/to/vendor/mypackage/foo.php"`
