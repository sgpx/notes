### Problem: Remove singleton dimensions from a tensor

**Objective:** Given a tensor with some dimensions of size 1, use `tensor.squeeze()` to remove those singleton dimensions.

---

### Step-by-step:

1. Create a 3D tensor with shape `(1, 3, 1)`:
```python
import torch
tensor = torch.randn(1, 3, 1)
print("Original tensor shape:", tensor.shape)
```

2. Use `squeeze()` to remove the singleton dimensions:
```python
squeezed_tensor = tensor.squeeze()
print("Squeezed tensor shape:", squeezed_tensor.shape)
```

3. Observe the difference before and after squeezing.

---

### Complete Code:

```python
import torch

# Create a tensor with singleton dimensions
tensor = torch.randn(1, 3, 1)
print("Original tensor shape:", tensor.shape)

# Remove singleton dimensions
squeezed_tensor = tensor.squeeze()
print("Squeezed tensor shape:", squeezed_tensor.shape)
```

---

### Explanation:
- The original tensor has shape `(1, 3, 1)`.
- After applying `.squeeze()`, it removes all dimensions with size 1, resulting in a shape `(3,)`.

This simple example helps you understand how to remove unnecessary singleton dimensions with `tensor.squeeze()` in PyTorch.
