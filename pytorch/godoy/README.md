# gradient descent

gradient descent is 
- a first order 
- iterative optimization algorithm
- for finding 
- the local minimum 
- of a differentiable function

# differentiable function

- a function whose derivative can be calculated at any point
- in visual terms, it should be smooth curves and should not have sharp edges

# gradient descent

iterative technique to

- find the best possible set 
- of parameters and cofficients
- for a given model, datapoints and loss functions
- starting from an initial guess
