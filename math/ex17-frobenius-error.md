# froebenius error

the froebenius error quantifies the difference between two matrices by calculating the square root of the sum of the absolute squared differences between all their elements

for A 

= 

a11 a12

a21 a22

and 

B

=

b11 b12

b21 b22

froebenius error = sqrt((b11-a11)**2 + (b12 - a12)**2 + (b21 - a21)**2 + (b22 - a22)**2)
