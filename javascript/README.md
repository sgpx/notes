# reduce example

```
> a = (x,y) => x + y;
[Function: a]
> [1,2,3].reduce(a)
6
```
# integer to binary string

```
const a = parseInt(Math.random() * Math.power(10,10));
console.log(a);

const b = a.toString(2);
console.log(b);
```

# integer to hex string

```
const a = parseInt(Math.random() * Math.power(10,30));
console.log(a);

const b = a.toString(16);
console.log(b);
```
