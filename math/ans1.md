# Scalars, units, and simple arithmetic review

**Question:** Given the following reported quantities from a training run, show all steps and reasoning:

A = 1200 g (mass)
B = 0.8 kg (mass)
C = 300 g (mass)
D = 45 s (time)
E = 0.5 min (time)
F = 300 K (temperature)
G = 3 m/s (speed)
H = 5 m east (displacement, vector)

1) Identify which of A–H are scalars. For each scalar you mark, give a one-line reason why it is a scalar (magnitude only).

2) Compute the total mass M_total = A + B + C and give your answer in kilograms. Show each conversion and arithmetic step with units.

3) Compute the total time T_total = D + E and give your answer in seconds. Show each conversion and arithmetic step with units.

4) Explain, using dimensional analysis, whether the expression M_total + T_total is a valid arithmetic sum. If it is not valid, state what must be done (if anything) to combine quantities of different dimensions.

**Answer:**

A,B,C,D,E,F,G are scalars because they don't have a direction and only have a magnitude. G is also a scalar because it only has magnitude and does not have a direction. H is a vector with both a magnitude and a direction.

M_total = 1200g + 0.8kg + 300g = 1.2kg + 0.8kg + 0.3kg =  2.3kg
T_total = 0.5 min + 45s = 30s + 45s = 75s

M_total + T_total is not a valid sum because M_total has dimension [M] while T_total has dimension [T] which means these two may not be added
