### **Problem: Learn to Add Two Numbers Using Different Data Types**

**Objective:**  
Create a small neural network (or just a function) that takes two input numbers, adds them together, and outputs the result. Demonstrate how data types like `torch.float32`, `torch.float64`, and `torch.float16` affect the computation.

---

### **Step-by-step guide:**

1. **Define two input tensors with different data types**

```python
import torch

# Define two tensors with default dtype (float32)
a_float32 = torch.tensor([1.5], dtype=torch.float32)
b_float32 = torch.tensor([2.5], dtype=torch.float32)

# Define tensors with float64
a_float64 = torch.tensor([1.5], dtype=torch.float64)
b_float64 = torch.tensor([2.5], dtype=torch.float64)

# Define tensors with float16
a_float16 = torch.tensor([1.5], dtype=torch.float16)
b_float16 = torch.tensor([2.5], dtype=torch.float16)
```

2. **Create a function that adds two numbers**

```python
def add_tensors(t1, t2):
    return t1 + t2
```

3. **Compute and print results with different dtypes**

```python
print("Addition with float32 tensors:", add_tensors(a_float32, b_float32))
print("Addition with float64 tensors:", add_tensors(a_float64, b_float64))
print("Addition with float16 tensors:", add_tensors(a_float16, b_float16))
```

4. **Observe how the dtype affects the precision and output**

---

### **Extra:**

- Try converting tensors to different data types using `.to(dtype)`.

```python
a = a_float32.to(torch.float64)
b = b_float32.to(torch.float64)
print("Converted to float64 and added:", add_tensors(a, b))
```
