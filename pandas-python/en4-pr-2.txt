Quick note: There isn’t a PyTorch function called pd.read_csv. pd.read_csv is from pandas. A typical workflow is to use pandas to read a CSV, then convert the data to PyTorch tensors for training.

Easy learning problem
- Task: Binary classification with two features x1 and x2.
- Label rule (easy for a model): y = 1 if x1 + x2 > 1, else y = 0.
- Why this is easy: A simple logistic regression (a single linear layer) can solve it well.

Self-contained script (generates a dataset, saves to data.csv, then loads with pd.read_csv and trains a small PyTorch model)
- It demonstrates:
  - Creating a CSV with pandas
  - Loading it back with pandas.read_csv
  - Converting to PyTorch tensors
  - Training a small logistic regression model
  - Evaluating accuracy on a test split

Code (copy-paste into a Python file and run)

```python
import os
import numpy as np
import pandas as pd
import torch
import torch.nn as nn
import torch.optim as optim

# 1) Generate a simple dataset and save to CSV (data.csv)
#    Label 1 if x1 + x2 > 1, else 0
N = 1000
rng = np.random.default_rng(seed=42)
X = rng.random((N, 2))  # x1, x2 in [0, 1)
y = (X.sum(axis=1) > 1.0).astype(int)

df = pd.DataFrame({'x1': X[:, 0], 'x2': X[:, 1], 'y': y})
data_path = 'data.csv'
df.to_csv(data_path, index=False)
print(f"Dataset saved to {data_path}")

# 2) Load the CSV with pandas read_csv
df_loaded = pd.read_csv(data_path)

# 3) Convert to PyTorch tensors
X_t = torch.tensor(df_loaded[['x1', 'x2']].values, dtype=torch.float32)
y_t = torch.tensor(df_loaded['y'].values, dtype=torch.float32).view(-1, 1)

# 4) Train-test split (80/20)
N = len(df_loaded)
perm = torch.randperm(N)
train_size = int(0.8 * N)

train_idx = perm[:train_size]
test_idx = perm[train_size:]

X_train, y_train = X_t[train_idx], y_t[train_idx]
X_test, y_test = X_t[test_idx], y_t[test_idx]

# 5) Create DataLoader objects
from torch.utils.data import TensorDataset, DataLoader

train_ds = TensorDataset(X_train, y_train)
test_ds = TensorDataset(X_test, y_test)

train_loader = DataLoader(train_ds, batch_size=32, shuffle=True)
test_loader = DataLoader(test_ds, batch_size=64)

# 6) Define a simple model (logistic regression)
class LogisticRegressionModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = nn.Linear(2, 1)

    def forward(self, x):
        return self.linear(x)  # logits

model = LogisticRegressionModel()

# 7) Loss and optimizer
criterion = nn.BCEWithLogitsLoss()
optimizer = optim.SGD(model.parameters(), lr=0.1)

# 8) Training loop
epochs = 50
for epoch in range(epochs):
    model.train()
    total_loss = 0.0
    for xb, yb in train_loader:
        logits = model(xb)
        loss = criterion(logits, yb)
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        total_loss += loss.item() * xb.size(0)
    avg_loss = total_loss / len(train_loader.dataset)
    if (epoch + 1) % 10 == 0 or epoch == 0:
        print(f"Epoch {epoch+1}/{epochs}, Loss: {avg_loss:.4f}")

# 9) Evaluation function (accuracy)
def evaluate(loader):
    model.eval()
    correct = 0
    total = 0
    with torch.no_grad():
        for xb, yb in loader:
            logits = model(xb)
            preds = torch.sigmoid(logits) >= 0.5
            correct += (preds.float() == yb).sum().item()
            total += yb.size(0)
    return correct / total

train_acc = evaluate(train_loader)
test_acc = evaluate(test_loader)

print(f"Training accuracy: {train_acc*100:.2f}%")
print(f"Test accuracy: {test_acc*100:.2f}%")
```

How to adapt
- Change the rule for y to create a different easy problem (e.g., y = 1 if x1 > 0.5 and x2 > 0.5).
- Increase dataset size for training stability.
- Extend to a small multi-class problem by adding a third feature and a label in {0,1,2}, and switch to a simple neural network with a final Softmax layer (or use CrossEntropyLoss with raw logits).
- If you want to test non-linear separability, you could modify the label to depend on a non-linear boundary (e.g., y = 1 if sin(pi*x1) > x2).

Note
- There is no PyTorch function named pd.read_csv. Use pandas.read_csv to load CSV data, then convert to PyTorch tensors as shown.
